# Format Code

Format Code makes it easy to format code snippets and add a theme for syntax highlight

More features coming soon, including support for more languages and improvements in usability.

## Languages supported

- CSS;
- HTML;
- Javascript;
- JSON;

## Themes supported

- A 11 Y Dark;
- A 11 Y Light;
- Atom One Dark;
- Atom One Light;
- Default;
- Dracula;
- Ocean;
- Paraiso Dark;
- Paraiso Light;
- Solarized Dark;
- Solarized Light;
- Vs;
- Vs 2015;
- Xcode;
